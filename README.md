# Leave_a_pic
ฝากรูปและแสดงลิงค์
รองรับไฟล์ PNG, JPEG, JPG, WEBP, MP4, MP3, WAV 

# วิธีนำโค้ดไปใช้
## โคลน repo นี้
```
git clone https://github.com/jirapongSrijanyoo/Leave_a_pic.git
```
เข้าไดเรทเทอรี
```
cd Leave_a_pic
```
ติดตั้งแพ็กเกจโดยใช้ [NPM](https://www.npmjs.com/)
```
npm i express multer
```
เริ่มรันโปรเจกต์
```
node index.js
```
